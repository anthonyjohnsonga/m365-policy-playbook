Resume saved work

Saved Excel files live in data\clients. Saving again updates the same file.

Import saved files

Bring in working files saved by another install of this app (e.g. the desktop copy on your work PC). Each file's playbook — Tier 1, Tier 0, Email Security — is detected automatically, and the client's folder is created if it doesn't exist yet. If the client already has a working file for that playbook it is replaced; the old file is backed up to _backups first.

Add policy to Tier 1 master

Adds a new policy to the Tier 1 master playbook. It appears in every new engagement created afterward (engagements already saved are unaffected). The master is backed up automatically before each change.
